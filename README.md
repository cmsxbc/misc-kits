# misc-kits

## Bookmarks

### generate_bookmarks.py

A tool to read bookmarks from browser, and render into html for share as web services.

#### supported browsers

- [x] chrome
- [x] firefox
- [x] chromium
- [ ] edge

#### features

- [x] path filter
- [x] icons(optional): retrieve real icon / random generated icon 
- [ ] title/url filters
